Stringsetting.async.contentType

概述[依赖jquery.ztree.core核心js]

Ajax提交参数的数据类型。[setting.async.enable=true时生效]

默认值："application/x-www-form-urlencoded"

String格式说明

contentType="application/x-www-form-urlencoded"可以满足绝大部分请求，按照标准的Form格式提交参数

contentType="application/json"可以满足.Net的编程需要，按照JSON格式提交参数

setting举例

1.设置Ajax提交参数的数据类型为JSON格式

varsetting={ async:{ enable:true, contentType:"application/json", url:"http://host/getNode.php", autoParam:["id","name"] } }; ......
