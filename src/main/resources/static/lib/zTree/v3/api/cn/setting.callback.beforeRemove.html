Function(treeId, treeNode)setting.callback.beforeRemove

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用于捕获节点被删除之前的事件回调函数，并且根据返回值确定是否允许删除操作

默认值：null

Function参数说明

treeIdString

对应zTree的treeId，便于用户操控

treeNodeJSON

将要删除的节点JSON数据对象

返回值Boolean

返回值是true/false

如果返回false，zTree将不删除节点，也无法触发onRemove事件回调函数

setting&function举例

1.禁止全部删除操作

functionzTreeBeforeRemove(treeId,treeNode){ returnfalse; } varsetting={ edit:{ enable:true }, callback:{ beforeRemove:zTreeBeforeRemove } }; ......
