Function(treeId,parentNode,responseData)setting.async.dataFilter

概述[依赖jquery.ztree.core核心js]

用于对Ajax返回数据进行预处理的函数。[setting.async.enable=true时生效]

默认值：null

Function参数说明

treeIdString

对应zTree的treeId，便于用户操控

parentNodeJSON

进行异步加载的父节点JSON数据对象

对根进行异步加载时，parentNode=null

responseDataArray(JSON)/JSON/String

异步加载获取到的数据转换后的Array(JSON)/JSON/String数据对象

v3.4开始支持XML数据格式的String

返回值Array(JSON)/JSON

返回值是zTree支持的JSON数据结构即可。

v3.x支持单个JSON节点数据进行加载
