String/Function(treeId,treeNode)setting.async.url

概述[依赖jquery.ztree.core核心js]

Ajax获取数据的URL地址。[setting.async.enable=true时生效]

默认值：""

String格式说明

设置固定的异步加载url字符串，请注意地址的路径，确保页面能正常加载

url内也可以带参数，这些参数就只能是通过get方式提交了，并且请注意进行转码

Function参数说明

treeIdString

对应zTree的treeId，便于用户操控

treeNodeJSON

需要异步加载子节点的的父节点JSON数据对象

针对根进行异步加载时，treeNode=null

返回值String

返回值同String格式的数据

setting&function举例

1.设置异步获取节点的URL为nodes.php

varsetting={ async:{ enable:true, url:"nodes.php", autoParam:["id","name"] } }; ......

2.设置异步获取节点的URL为function动态获取

functiongetAsyncUrl(treeId,treeNode){ returntreeNode.isParent?"nodes1.php":"nodes2.php"; }; varsetting={ async:{ enable:true, url:getAsyncUrl, autoParam:["id","name"] } }; ......
